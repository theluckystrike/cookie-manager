# Cookie Manager - Screenshot Specifications

All screenshots should be 1280x800 PNG format.

## Screenshot 1: Main View
**Headline:** "See All Your Cookies at a Glance"
**Content:** Show the popup with a list of 8-10 cookies for a site like "github.com"
**Background:** Dark purple gradient (#0f0f1a to #1a1a2e)
**Elements:**
- Cookie Manager popup centered
- Cookies visible with badges (Secure, HttpOnly, JWT)
- Search bar visible with "12" cookie count

## Screenshot 2: Cookie Editor
**Headline:** "Edit Cookies with Ease"  
**Content:** Show the edit modal open with a cookie being edited
**Background:** Dark purple gradient
**Elements:**
- Editor modal visible
- All fields populated
- JWT badge visible if editing a JWT cookie

## Screenshot 3: JWT Decoder
**Headline:** "Decode JWT Tokens Instantly"
**Content:** Show the JWT decoder modal with decoded header and payload
**Background:** Dark purple gradient
**Elements:**
- JWT modal visible
- Header and payload JSON visible
- Expiration status showing
