# Changelog

All notable changes to Cookie Manager will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-24

### Added
- View all cookies for current site
- Edit cookie name, value, domain, path, and expiration
- Delete individual cookies
- Clear all cookies for a domain
- Export cookies as JSON (copy to clipboard)
- JWT token detection and decoding
- Read-only mode toggle for protection
- Dark mode support (follows system preference)
- Right-click context menu for quick clear
- Keyboard shortcuts (/ for search, Escape to close)

### Security
- All data stays local - no network requests
- No analytics or tracking
- Open source for transparency
