Cookie Manager Help

Getting Started

What is Cookie Manager?

Cookie Manager is a browser extension that lets you view, edit, delete, export, and import cookies for any website. It provides a clean interface with security insights like JWT detection and flag visibility.

How do I open Cookie Manager?

Click the Cookie Manager icon in your browser toolbar. If you do not see it, click the extensions puzzle-piece icon and pin Cookie Manager for quick access.

What should I do first?

Navigate to any website, then open Cookie Manager. You will see all cookies for the current domain listed automatically. Use the search bar to filter cookies by name or value.

Managing Cookies

How do I view cookies?

Open Cookie Manager on any page. All cookies for the current domain are displayed in a scrollable list. Each cookie shows its name, a truncated value, and badges for security flags like Secure, HttpOnly, and JWT.

How do I edit a cookie?

Click on any cookie in the list to open the edit modal. You can modify the name, value, domain, path, expiration, and security flags. Press Save to apply changes. Note: editing is disabled in read-only mode.

How do I delete cookies?

Open the edit modal for a cookie and click the Delete button. You can also use the bulk delete shortcut Ctrl+Shift+D to remove all cookies for the current domain.

How do I export cookies?

Use the export button in the action bar or press Ctrl+E. Cookies are exported as a JSON file that can be saved and shared. The export includes all cookie attributes.

How do I import cookies?

Click the import button in the action bar. Select a previously exported JSON file. Cookie Manager will validate the file and import the cookies, notifying you of any that failed.

Security Features

What is read-only mode?

Read-only mode prevents accidental edits or deletions. When enabled, cookie values can be viewed and copied but not modified. Toggle it using the lock switch in the header.

What do the Secure and HttpOnly flags mean?

Secure means the cookie is only sent over HTTPS connections. HttpOnly means the cookie cannot be accessed by JavaScript, protecting against XSS attacks. These flags are shown as colored badges on each cookie.

What is JWT detection?

Cookie Manager automatically detects cookies containing JSON Web Tokens (JWTs). These are marked with a JWT badge. Click the badge to decode and inspect the token header, payload, and expiration status.

Troubleshooting

The extension is not loading

Try disabling and re-enabling the extension in your browser's extension settings. If the problem persists, remove and reinstall it. Make sure your browser is up to date.

Cookies are not showing up

Ensure you are on a website (not on a browser internal page like chrome://). Some sites set cookies only after interaction. Try refreshing the page, then reopening Cookie Manager.

Export is not working

Check that you have at least one cookie listed. If the download does not start, your browser may be blocking it -- check the downloads bar or popup blocker settings. Try using a different export format if available.

The extension feels slow or laggy

Sites with a very large number of cookies can cause slowdowns. Use the search bar to filter the list. You can also try closing other extensions that might conflict. Use the Quick Diagnostics button above for more information.
