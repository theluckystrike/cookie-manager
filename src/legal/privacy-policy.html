Privacy Policy

Cookie Manager by Zovo

Last updated: February 11, 2026

1. Introduction

Cookie Manager is a Chrome extension developed by Michael Ip / Zovo (zovo.one). This Privacy Policy explains how Cookie Manager handles your data. In short: your data stays on your device. We do not collect, transmit, or share any personal information.

This policy applies to all versions of the Cookie Manager browser extension. By using Cookie Manager, you acknowledge the practices described in this document.

2. Quick Summary

Cookie Manager is a 100% local extension. It makes zero external network requests, has no user accounts, and shares nothing with third parties.

3. Information We Collect

Cookie Manager stores the following data locally on your device using chrome.storage.local. None of this data is transmitted externally.

  • Settings preferences — Your configuration choices such as read-only mode state and protected domain lists.
  • Anonymous usage counters — Local-only counts of feature usage (e.g., how many times you exported cookies) to help improve the interface.
  • Error logs — Debugging information stored locally to help diagnose issues. Limited to 50 entries with automatic rotation.
  • Popup performance metrics — Load time measurements stored on your device to monitor extension performance.

4. Information We Do NOT Collect

Cookie Manager explicitly does not collect, process, or transmit:

  • Personal information (names, emails, phone numbers, addresses)
  • Browsing history or page visit data
  • Cookie contents or values to any external server
  • Form data or passwords
  • IP addresses
  • Device fingerprints or hardware identifiers
  • Location data
  • Any data via analytics services, telemetry, or tracking pixels

Cookie Manager does not set any cookies itself. It only reads and manages cookies that websites have already placed in your browser.

5. How We Use Information

The locally stored data described in Section 3 is used solely to:

  • Provide core extension functionality (viewing, editing, exporting, and importing cookies)
  • Remember your settings and preferences between sessions
  • Track feature usage locally so we can improve the interface in future updates
  • Store error logs on your device for debugging if you encounter issues
  • Measure popup load performance for optimization purposes

All processing happens entirely within your browser. No data leaves your device.

6. Data Sharing

We do not share any data with anyone. Period.

Cookie Manager makes zero external network requests. There are no analytics services, no advertising networks, no third-party SDKs, and no server-side components. Your data cannot be shared because it never leaves your device.

7. Data Storage & Security

All extension data is stored in chrome.storage.local, which is:

  • Sandboxed — Isolated by Chrome's built-in extension security model. Other extensions and websites cannot access this data.
  • Local only — Stored on your device's file system, never synced to external servers.
  • User-controlled — You can clear all extension data at any time through Chrome's extension settings or through Cookie Manager's built-in tools.

The extension does not use any external databases, cloud storage, or remote servers.

8. Your Rights

You have full control over all data stored by Cookie Manager:

  • Access — View all stored settings and data through the extension's interface.
  • Export — Export your cookie data using the built-in export feature.
  • Delete — Remove any or all stored data through the extension or via Chrome's "Clear storage" option in the extension management page.
  • Uninstall — Removing the extension automatically deletes all associated local data.

Since we do not collect or store any data on external servers, there is no remote data to request or delete.

9. Data Retention

Locally stored data follows these retention limits:

  • Error logs — Maximum of 50 entries, automatically rotated (oldest entries are removed when the limit is reached).
  • Analytics events — Maximum of 100 events stored locally, automatically rotated.
  • Settings — Persisted until you change or reset them.
  • All data — Permanently deleted when the extension is uninstalled.

10. Children's Privacy

Cookie Manager does not collect personal data from any user, regardless of age. Because no personal information is collected, stored externally, or transmitted, the extension does not pose any privacy risk specific to children.

11. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be delivered through extension updates distributed via the Chrome Web Store. The "Last updated" date at the top of this page will be revised to reflect any modifications.

We encourage you to review this policy periodically. Continued use of Cookie Manager after changes constitutes acceptance of the updated policy.
