2. Description of Service

Cookie Manager is a free browser extension that allows users to manage HTTP cookies for websites they visit. The Extension operates entirely on your local device and makes zero external network requests. Key features include:

  • Viewing, creating, editing, and deleting cookies for the current site
  • Exporting cookies in JSON and Netscape formats
  • Importing cookies from exported files
  • Automatic JWT (JSON Web Token) detection and decoding
  • Search and filter functionality across cookies
  • Read-only mode to prevent accidental modifications
  • Domain protection settings
  • Keyboard shortcuts for efficient workflow

Cookie Manager is provided at no cost. There are no paid features, subscriptions, or in-app purchases.
